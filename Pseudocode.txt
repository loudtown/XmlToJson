1. Получить входные параметры (исходный файл, имя записываемого файла)

2. Проверить, существует ли записываемый файл. Если существует - уточнить, переписывать ли его (если не переписывать - нет смысла конвертировать, программа завершается)

3. Если всё хорошо - чтение файла

4. Если всё хорошо - конвертация содержимого файла

4.1 Убрать XML-Declaration

4.2 Преобразовывать элементы

4.2.1 Получить первый элемент

4.2.2 Проверить, единственный ли это элемент такого рода (с таким именем) на данном уровне вложенности. Если нет, то он часть какого-то массива.

4.2.3 Для содержимого каждого из элементов такого рода применить 4.2, разделяя результаты запятой

4.2.4 Если это часть массива - заключить результат обработки содержимого в [ ]. Иначе - применить 4.2 к оставшемуся тексту на данном уровне

4.2.5 Если элементов на данном уровне было несколько - обернуть результат в { }

4.2.6 Если родитель элементов данного уровня не был элементом массива, поместить его имя в кавычках перед результатом

4.2.7 Если элемента изначально получено не было - проверить, имеются ли данные в содержимом. 

4.2.8 Если имеются, результатом будут эти данные в кавычках

4.2.9 Если это не элемент массива, поместить имя элемента в кавычках перед результатом.

4.3 Убрать лишние запятые (после последних элементов объектов, массивов)

5. Если всё хорошо - запись полученной строки в файл
